Privacy Policy

Data Collection

DiscordTrack collects only the necessary information required to provide our tracking services:

  • Discord User IDs
  • Server announcement data
  • Public mention history
  • User preferences and settings

Data Usage

Your data is used exclusively for:

  • Tracking announcements in your selected servers
  • Monitoring mentions of specified users
  • Generating analytics and reports
  • Improving our services

Data Protection

We implement industry-standard security measures:

Storage Security

All data is encrypted at rest using AES-256 encryption

Transmission Security

Data in transit is protected using TLS 1.3
